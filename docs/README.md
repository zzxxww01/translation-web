## 📝 文档维护原则

1. **单一职责**：每份文档只聚焦一个主题，避免内容重复
2. **相互引用**：通过链接引用其他文档，而不是复制内容
3. **及时更新**：代码变更后及时更新相关文档的更新时间
4. **真实准确**：文档描述必须与当前代码实现一致
5. **分层清晰**：系统架构 → 链路文档 → 技术手册 → 模块手册
