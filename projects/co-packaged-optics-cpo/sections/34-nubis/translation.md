2025年10月，Nubis 刚刚被 Ciena acquired 。与 Ayar 类似，Nubis 提供光引擎（Optical Engine，OE）芯粒以集成到客户的 host silicon 中，但重点在于单波长连接。Nubis 专注于互操作性——包括协议和机械（即可插拔）互操作性，这也决定了其技术选择。Nubis 还有更广泛的使命，即解决 I/O 墙问题，其解决方案同时涵盖光学和铜缆。

**修订后的译文：**

他们现有的光引擎产品是 Vesta 100 1.6T NPX 光引擎。这是一个可插座式模块，可提供 1.6T 双向带宽，包含 16 条 100G 通道。该模块的尺寸为 6×7 mm。与其他公司不同，Nubis 大量采用马赫-曾德尔调制器（Mach-Zehnder Modulator，MZM），主要原因是该调制器具有良好的互操作性、可靠性和成熟度。另一个主要设计选择是，Nubis 兼容 IEEE/OIF 标准规定的电气接口，因为他们认为大多数 ASIC 开发商将继续使用这些技术。

Nubis 的一个关键差异化点在于其光纤耦合方式。Nubis 从 PIC 表面进行耦合，具体采用一块薄玻璃来帮助布线和对齐光纤。与将光纤连接到芯片边缘的边缘耦合不同，Nubis 的 2D 光纤阵列方案是从硅光子芯片顶部连接光纤。

观察下图：PIC（底部绿色部分）包含调制器、光探测器和波导，其上方安装有 EIC。红色柱体为光纤，而容纳光纤的块状结构是玻璃块（FAU），用作光纤固定器。FAU 在块体顶部采用激光钻孔，以确保光纤精确定位。通过采用 2D 光纤阵列，他们能够将 36 根光纤（16 根发送、16 根接收、4 根激光器）连接到 PIC，并避免使用 WDM 在更少光纤上传输更多波长。这使得 Nubis 的 FAU 成为目前已出货产品中密度最高的光纤阵列之一。

2D 光纤阵列已被纳入台积电等公司的路线图，也是垂直耦合的一大优势。不过，目前除 Nubis 之外尚无其他公司实现量产，这使其脱颖而出，尽管其他厂商计划后续转向 2D 阵列。

光纤向上引出后，通过采用住友电工开发的特殊光纤 FlexBeamGuidE 实现 90 度弯曲侧向引出，该光纤可在 90 度弯曲状态下仍保持高可靠性和低损耗。

采用 2D 阵列而非边缘耦合的另一个优势是，在物理上受可连接光纤数量的限制更小。如图所示，使用 Nubis 的 2D 光纤阵列结构，可以在 ASIC 周围布置多排光引擎，只要封装允许，即可显著提升带宽密度。

2025 年 4 月，Nubis 宣布其下一代 PIC 量产，该硅光子芯片每通道支持 16×200G，单向前沿带宽密度达到 0.5Tbps/mm（与电气 host 接口密度相当）。此外，Nubis 宣布与 Samtec 合作，将提供 32×200G（6.4T）光模块样品，该模块可直接卡入 Samtec Si-Fly HD 共封装铜缆连接器。与其他共封装光学（Co-packaged Optics，CPO）方案相比，这种方法实现了铜缆与光学的通用封装尺寸，长期来看有可能形成一个开放的可插拔生态系统，用于部署 CPO。

**修订后的译文：**
最后，在铜缆领域，Nubis 也在 OFC 上宣布并展示了其线性重驱动器芯片 Nitro，用于有源铜缆（ACC），可将 200G 铜缆的传输距离延伸至数米。这一方案是与安费诺（Amphenol）合作开发的，后者将基于 Nitro 线性重驱动器芯片生产 ACC。
