### H100租赁价格飙升40%，GPU租赁定价仪表盘上线，算力租赁市场结构解析，租赁价格还会继续上涨吗？

作者：Dylan Patel、Daniel Nishball、Jordan Nanos 等2人

2026年4月2日 · 免费

Anthropic 的 Claude 4.6 Opus 和 Claude Code 需求激增。短短一个季度内，Anthropic 的年度经常性收入（ARR）几乎翻了三倍，从去年底的 90 亿美元飙升至如今的 250 亿美元以上。GLM 和 Kimi K2.5 等开源模型也推动了开源模型应用场景的暴涨。此外，Anthropic、OpenAI 以及各类新兴 AI 实验室（Neolab）完成融资后，同样产生了对 GPU 的庞大需求。

- Claude Code 是行业拐点 - Doug O'Laughlin、Jeremie Eliahou Ontiveros 等3人 · 2月5日

这一拐点意味着需求已然暴增，超大规模云厂商和新兴算力云厂商（Neocloud）的 GPU 遭遇抢购潮。

这股新增需求推高了整个供应链的产品与服务定价，从 DRAM 和 NAND 存储器，到光纤线缆、数据中心托管乃至燃气轮机，无一幸免。

在众多算力相关的产品与服务中，GPU 租赁是最新一个因供应极度紧张而导致价格跳涨的领域。H100 的 1 年期 GPU 租赁合同价格已从 2025 年 10 月 1.70 美元/小时/GPU 的低点，暴涨近 40%，至 2026 年 3 月的 2.35 美元/小时/GPU。

所有型号的按需 GPU 租赁算力均已售罄——那些已经锁定按需实例的客户，尽管面临近期价格跳涨，也不愿将这些算力释放回资源池。在2026年初寻找 GPU 算力，就像试图预订最后一班航班的机票一样：价格高昂，且几乎一票难求。这还是个体面的比喻，更准确的说法是，现在想租一个集群简直像在买毒品一样。

得益于我们在ClusterMAX、InferenceX以及AI 云总拥有成本方面的工作，SemiAnalysis 一直在深入追踪新兴算力云厂商和超大规模云厂商生态内的 GPU 租赁定价等趋势与议题。

我们还投入大量时间协助 AI 实验室对接新兴算力云厂商，在市场上寻找 GPU 租赁资源，并与生态内的几乎所有参与者积极探讨 GPU 租赁价格趋势。

自2023年以来，我们一直为客户维护着GPU 租赁价格指数，追踪大多数主流 GPU 型号（H100、H200、B200、B300、GB200、GB300、MI300、MI325、MI355）在所有关键合同期限下的定价，涵盖从按需、1个月一直到5年的各类期限。我们的指数基于对众多新兴算力云厂商和算力买家的调研数据构建，并通过实际交易数据，以及我们亲自促成的谈判与交易进行了交叉验证。
