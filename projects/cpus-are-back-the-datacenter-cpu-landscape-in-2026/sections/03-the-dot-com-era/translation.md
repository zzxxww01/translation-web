21 世纪初迎来了互联网时代。Web 2.0、电子邮件、电子商务、谷歌搜索以及支持 3G 宽带数据的智能手机相继涌现。随着万物触网，数据中心 CPU 必须承担起处理全球互联网流量的重任，并随之成长为一个价值数十亿美元的细分市场。在设计层面，随着登纳德缩放定律走向终结，“主频大战”偃旗息鼓，业界的注意力转向了多核 CPU 与更高的集成度。AMD 将内存控制器直接集成到 CPU 裸片中，高速 I/O（PCIe）也改为直接由 CPU 提供。多核 CPU 格外契合数据中心工作负载，能让大量任务在不同核心上并行处理。

我们将在下文的互连架构章节详细剖析多核连接方式的演进历程。AMD 和英特尔也在这一时期引入了同步多线程 (SMT) 技术。该技术将一个物理核心划分为两个逻辑线程，两者在共享大部分核心资源的同时能够独立运行，从而进一步提升了可并行化数据中心工作负载的性能。对性能有更高要求的用户则会转向多路 CPU 服务器。英特尔的快速通道互联 (QPI) 与 AMD 皓龙 CPU 采用的 HyperTransport直连架构，能够在单台服务器中为多达八个插槽提供一致性链路。
